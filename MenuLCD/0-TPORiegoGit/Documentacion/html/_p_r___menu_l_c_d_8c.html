void Menu_LCD ( uint8_t  TeclaPulsada)

Función que permite utilizar el LCD a modo de menú.

Esta función recoge los datos de humedad y temperatura para mostrarlos en un LCD de manera interactiva mediante diferentes pantallas (escrituras) en el LCD, las cuales pueden ser recorridas mediante las teclas pulsadas en el teclado 5x1. Además se permite la configuración de horas automáticas de riego, de la hora, la fecha y valores críticos.
